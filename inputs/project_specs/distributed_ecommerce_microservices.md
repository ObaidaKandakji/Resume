# Project Spec: Distributed E-Commerce Microservices Platform

## Resume Title

Distributed E-Commerce Microservices Platform

## Repositories / Links

- https://github.com/ObaidaKandakji/CST-8915-Final-Project

## Project Type

Cloud-native microservices platform.

## Problem It Solves

Implements an e-commerce workflow across distributed services with asynchronous order processing and deployment automation.

## What Was Actually Built

- 2 Vue.js frontends and 3 backend services deployed to Azure Kubernetes Service.
- Event-driven order flow using REST APIs, RabbitMQ, and MongoDB.
- CI/CD pipeline for 5 services with Docker and GitHub Actions.
